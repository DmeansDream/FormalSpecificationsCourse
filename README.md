# Dafny
---

## Коренева папка
В корені проєкту міститься код для Системи турнікету метро виконаний у Dafny. Код міститься у файлах: Program.dfy, CardClasses.dfy, Turnstile.dfy.
Також тут знаходиться файл Program.cs, який є автоматичною трансляцією коду Dafny в еквівалентний код C#, виконаною самою мовою Dafny.

## CSharpRewrite
Директорія містить код мовою С# який був написаний вручну для переведення Системи створеною мовою Dafny у максимально еквівалентний код іншою мовою ООП високого рівня.

## AI Coding
Директорія містить код Системи створений виключно з використанням двох моделей штучного інтелекту з мінімальним втручанням людини. В подальшому даний код було проаналізовано за допомогою експертної системи LLM as-a-judge використовуючи бінарне порівняння між собою та з кодом написаним людиною.

## Programming Dafny
Директорія містить матеріали для реферату #2 з прикладами реалізації Solid та патернів програмування Singleton та Flywheel в Dafny.
